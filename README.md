FUTURE_CS_02 - Phishing Detection & Awareness


Overview

This part Demonstrates how phishing emails are detected using header analysis and HTML link Inspection and it provides a clear awareness guidlines to help employees to avoid phishing attacks.

Objectives

1. analyze real phishing email samples
2. identify common phishing indicator
3. classfify email risk levels
4. explain attacks in simple, non-technical language
5. provide prevention tips and wareness guidelines

Tools used

1. Google message header analyser to interpret email metadata
2. MXtoolbox header analyser to verify SPF, DKIM, DMARC records
3. Text editor( Notepad) to inspect raw HTML safely

Key Findings

1. Bradesco livelo > spoofed banking domain, failed SPF/DKIM/DMARC and urgency language
2. Microsoft Alert > fake security domain, mailto, Redirect, authentication failures
3. Solar Panel Offer > unverified sender, generic promotion , no authentication records

Awareness Guidlines

Do's: verify sender domains , hover links, report suspicious email, use MFA
Don;ts: click unknown links, Share passwords/OTPs , ignore warnings
